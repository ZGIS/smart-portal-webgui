Manage uploaded files

Uploading a file automatically creates an entry into your Default Collection. You are able to share this file with others users by adding it to another collection and sharing this collection with the user(s).

To make your file publicly discoverable and accessible, you will need to create a metadata record within "My Metadata" and link this metadata to your file by typing the filename in the "Distribution Link" input field during metadata record creation.
